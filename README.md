# RxInsight Compliance Auditor

An AI-powered regulatory compliance auditor specialising in 
South African pharmaceutical legislation and pharmacy practice standards.

**Live App:**  https://partyrock.aws/u/ChinonyeOkolie/HcZm0BGUC/RxInsight-Compliance-Auditor

## What It Does
Audits marketing materials, advertisements, promotional claims, 
and patient communications against:

- Medicines Act 101 of 1965
- SAHPRA advertising requirements
- SAPC ethical standards
- Good Pharmacy Practice (GPP)
- Marketing Code Authority (MCA) Code of Marketing Practice

## Output
Generates a structured compliance report identifying:
- Regulatory risks
- Advertising breaches
- Ethical concerns
- Patient-safety risks

## Why I Built This
As a pharmacy, I identified a gap - pharmaceutical marketers and healthcare communicators in SA 
have no quick way to self-audit content before publication. 
This tool addresses that.
